Добро пожаловать

Это приложение разработано для визуализации некоторых сортировок массива, и изначально разработано как творческое задание по предмету "программирование графики"

Гайд

Сверху отображена статистика по массиву и текущей сортировке.
Правее можно найти переключатель типа отображения:

VECTOR - Отображет массив как элементы HTML, CSS в форме векторной графики
CANVAS - Реализация за счёт стандартного элемента HTML Canvas, который позволяет рисовать примитивные растровые формы

Ниже, находится окно отображения массива, данные выводятся как график, в котором каждый столбец соответствующий элемент массива.
Далее, идёт раздел управления сортировкой, 3 кнопки управления авто-сортировкой:
  • Сбросить сортировку
  • Остановить сортировку (пауза)
  • Запустить авто сортировку
Кнопка ручной пошаговой сортировки
И настройки в виде слайдеров, где можно установить:
  • скорость шага (для автосортировки)
  • Скорость свойства плавности анимации (только для VECTOR типа)
Ниже под кнопкой пошаговой сортировки, находится поле ввода массива, числа вводятся через запятую, и что-бы отобразить массив, необходимо нажать клавишу Enter или кнопку справа
Слева находится раздел генерации рандомного массива, кнопка которая генерирует случайный массив, и кнопка настроек генерации, где можно указать длинну, максимальный и минимальный элементы массива

И последний раздел это раздел выбора типа сортировки.

Рекомендации:

Регулируйте свойства скорости шага и плавности анимации для лучшего отображения.
Если текущий элемент плохо выделяется, или не выделяется вовсе, поставьте плавность анимации близкую к 0 или 0
